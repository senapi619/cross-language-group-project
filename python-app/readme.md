# Book Cataloging System

## Overview

The Book Cataloging System is a GUI-based desktop application developed in Python using Tkinter.  
It allows users to manage a collection of books by adding, removing, searching, and generating reports by author or genre.

This project demonstrates Python's dynamic typing, list-based data storage, and event-driven GUI design.

---

## Features

- Add books with:
  - Title
  - Author
  - Genre
  - Publication Year

- Remove books:
  - By selecting from the catalog grid
  - By entering the book title

- Search books by:
  - Title
  - Author
  - Genre

- Display all books in the catalog

- Generate reports grouped by:
  - Genre
  - Author

- Preloaded sample data:
  - Harry Potter
  - 1984
  - Dune

- Books accumulate in the grid as they are added

## Running the Application

1. Open the project folder in your IDE or terminal.
2. Navigate to the `python-app/src` directory.
3. Run the application using:

```bash
python main.py
```

Once executed, the Book Cataloging System graphical interface will open.

## How to Use

### Add a Book

1. Enter:
   - Title
   - Author
   - Genre
   - Publication Year
2. Click **Add Book**

The book will immediately appear in the catalog grid.

---

### Remove a Book

Option 1:
1. Select a book from the catalog grid
2. Click **Remove Book**

Option 2:
1. Enter the title in the Title field
2. Click **Remove Book**

---

### Search for Books

Use the search buttons to search by:

- Title
- Author
- Genre

Search results replace the current catalog view.

---

### Display All Books

Click **Display All** to show all books in the catalog, including preloaded sample books.

---

### Generate Reports

Reports can be generated by:

- Clicking **Report Genre**
- Clicking **Report Author**

A new window opens displaying books grouped accordingly.

The following books are preloaded into the system:

- Harry Potter
- 1984
- Dune

Important behavior notes:

- Sample books are NOT displayed at startup
- They appear only after clicking **Display All**
- Searches and reports include all books:
  - Sample books
  - User-added books

---

## Catalog Behavior

- The catalog grid accumulates books as users add them
- Newly added books remain visible in the running list
- Clicking **Display All** combines:
  - Preloaded books
  - User-added books
