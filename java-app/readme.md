# Book Cataloging System 

## Overview

The Book Cataloging System is a GUI-based desktop application developed in Java using Swing.  
It allows users to manage a collection of books by adding, removing, searching, and generating reports by author or genre.

This project demonstrates object-oriented programming principles and event-driven GUI design.

## Features

- Add books with title, author, genre, and publication year
- Remove books by title
- Search books by title, author, or genre
- Display all books in the catalog
- Generate reports grouped by genre or author
- Preloaded sample data for demonstration

## Project Structure

Book-Cataloging-System/

- Book.java → Represents a Book object  
- BookManager.java → Handles all logic (add, remove, search, reports)  
- MainGUI.java → Java Swing graphical user interface  
- Main.java → Program entry point  
- README.md → Project documentation  

## Requirements

To run this project, Java must be installed on the system. The recommended version is JDK 17 or higher. The project can be executed using any Java IDE such as IntelliJ IDEA or Visual Studio Code. If using VS Code, the Java Extension Pack should be installed.

## Running the Application

To run the program, first open the project folder in your IDE or in Visual Studio Code. Once the folder is opened, open a terminal in the project directory. The project is compiled by running the command:

javac *.java

This command compiles all Java source files in the folder. After successful compilation, the application can be started using the command:

java Main

Once executed, the graphical user interface for the Book Cataloging System will open and the user can begin interacting with the application.

## How to Use

To add a book, the user enters the title, author, genre, and publication year in the input fields and clicks the Add Book button.  
To remove a book, the user enters the title and clicks Remove Book.  
To search for books, the user can search by title, author, or genre using the respective buttons.  
To view all books, the Display All button can be used.  
Reports can be generated by clicking Report Genre or Report Author, which groups books accordingly.
